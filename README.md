# Unity Slot Machine Game

## Overview

This project is a slot machine game developed in Unity as part of a technical assessment. The game simulates a classic slot machine experience with animated spinning reels, randomized outcomes using RNG, win detection, and payout calculation.

## Features

* Randomized reel outcomes using Unity RNG
* Smooth reel spin animations
* Win detection when all reels display the same symbol
* Payout calculation based on winning combinations
* Clean and modular OOP-based architecture
* Responsive UI and game feedback
* WebGL build included

## Controls

1. Press the **Spin** button.
2. Wait for all reels to stop spinning.
3. If all symbols match, the player wins and receives a payout.
4. Press **Spin** again to continue playing.
